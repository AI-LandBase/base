# Base

あらゆる新規プロジェクトの上流に置く、用途非依存のスターター足場。アプリコードやフレームワーク依存物は持たず、AI/人間の協働インフラ (`.claude/`、`CLAUDE.md`、`CONTRIBUTING.md`、`.github/`) だけを提供する。コード案件・非コード案件 (調査、マーケティング等) を問わず使える。

## Language

**Base**:
このリポ。案件種別 (Web、CLI、マーケティング、調査等) もフレームワーク有無も問わず、新規プロジェクト着手時に最初に clone する土台。協働足場のみを含み、特定技術への依存を持たない。
_避ける_: Template, 雛形, スキャフォールド, スターター (この成果物は「Base」で統一)
